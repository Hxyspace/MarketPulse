# 市场仪表盘 Money Dashboard

基于公开数据的量化市场监控面板，包含三大核心指标模块。

## 功能模块

### 01 红利罗盘
- **指标**：红利低波（H30269）相对中证全指（000985）的 40 日收益差
- **逻辑**：收益差 > +10% 过热减仓，-1%~+10% 正常持有，< -1% 过冷加仓
- **数据源**：中证指数官方 API + 腾讯财经 API

### 02 债市晴雨表
- **指标**：中债新综合净价指数温度（历史百分位）+ 每日天气
- **逻辑**：温度 0-100℃，< 30℃ 低位、30-80℃ 正常、> 80℃ 高位
- **数据源**：中国债券信息网

### 03 基金温度计
- **指标**：沪深 300 股债利差（Equity Risk Premium）
- **逻辑**：股债利差 = 1/PE × 100 - 10 年国债收益率，取近 10 年百分位映射为温度
- **温度区间**：≤ 30℃ 低估（加倍定投）、30-80℃ 正常（按计划定投）、> 80℃ 过热（止盈）
- **数据源**：中证指数 PE 数据 + 中国债券信息网国债收益率

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建 & 启动
npm run build
npm start
```

访问 http://localhost:3000

## 飞书推送配置

使用飞书应用机器人（Open API）推送消息，需要在飞书开放平台创建应用并开启机器人能力。

1. 复制 `.env.example` 为 `.env`
2. 填入应用凭证和群聊 ID：

```bash
FEISHU_APP_ID=cli_xxxx        # 应用 App ID
FEISHU_APP_SECRET=xxxx         # 应用 App Secret
FEISHU_CHAT_ID=oc_xxxx         # 目标群聊 chat_id
```

3. 确保应用已开启机器人能力，且机器人已加入目标群聊
4. 应用需要 `im:message:send_as_bot` 权限

配置后每个交易日 18:00（北京时间）自动推送市场速报。

## 技术栈

- **运行时**：Node.js + TypeScript
- **框架**：Express
- **图表**：ECharts
- **定时任务**：node-cron
- **数据存储**：本地 JSON 文件缓存（自动增量更新）

## 数据源

| 数据 | 来源 | 说明 |
|------|------|------|
| 指数行情 | 中证指数官方 API | 红利低波、中证全指、沪深300（含 PE） |
| 指数行情（备用） | 腾讯财经 API | 中证全指等（无 PE） |
| 债券净价指数 | 中国债券信息网 | 中债新综合净价指数（2002年至今） |
| 国债收益率 | 中国债券信息网 | 10年期国债到期收益率（2010年至今） |

## 数据更新逻辑

- 北京时间 20:00 前：取前一个交易日数据（数据源通常在晚间更新）
- 北京时间 20:00 后：取当天数据
- 每个最新交易日只拉取一次，本地 JSON 缓存 + 增量更新
- 中债数据时间戳为北京时间，存储时已做 UTC+8 还原
